## `matterhorn_pytorch.tnn.functional` ： 时空代数的算子

该模块存储了 Temporal Neural Networks （TNNs）的基础——时空代数（Space-time Algebra）的 10 个运算符。除此之外，其还存储了一些 TNNs所用到的函数。

您可以阅读[文档](./1_functional.md)以了解更多函数的使用方法。
